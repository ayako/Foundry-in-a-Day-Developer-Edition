## 各 Lab の進め方

1. Lab の **README.md** を上から読み、手順 (Step) に沿ってコマンドを実行
2. 各 Step の最後にある **成功確認** を満たしているかチェック
3. つまずいたら **よくあるエラー** または [../troubleshooting.md](../troubleshooting.md)
4. 時間が余ったら **Challenge** に挑戦

## 執筆者向け

- 各 Lab の README は [../lab-template.md](../lab-template.md) から写経して作成しています
- 完成形コードは [../solutions/](../solutions/) に配置してください
- スクリーンショットは各 Lab の `assets/` ディレクトリに配置
- 参加者がコピペ実行するコードは各 Lab の `src/` ディレクトリに配置
