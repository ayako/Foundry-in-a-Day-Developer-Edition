# solutions/

各 Lab の **完成形コード** を配置するディレクトリです。

## 配置ルール

- ディレクトリ構成は `labs/` に揃える: `solutions/lab1-sdk-cli/`, `solutions/lab2-agent-framework/`, ...
- Lab の `src/` に置くコードと **同じファイル名** で配置 (差分が一目でわかるように)
- secret や個人の Foundry エンドポイントを含めないこと

## 公開範囲

- **参加者にも公開** (Lab で詰まったときの答え合わせ用)
- 講師は Lab 説明時に answer key として参照可

## 執筆者へ

- Lab 中身が確定するまで空でかまいません
- 答えを置く際は「最小限の動く例」にとどめ、過度な抽象化は避けてください
