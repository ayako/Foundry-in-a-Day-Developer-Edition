# scripts/

ワークショップ全体で共通で使うスクリプト置き場です。

## 想定する用途

- 環境構築の自動化 (Azure リソースのプロビジョニング等)
- 動作確認スクリプト (`check_setup.ps1` / `check_setup.sh` など)
- Lab 横断のデータ準備

## 命名規則

- PowerShell: `*.ps1`
- Bash: `*.sh`
- Python: `*.py`
- 同じ機能の OS 別スクリプトは **同じファイル名のサフィックス違い** で並べる

## 公開範囲

- 参加者にも公開

## 執筆者へ

- 必要が出たタイミングで追加してください
- secret や API キーを直書きしないこと
